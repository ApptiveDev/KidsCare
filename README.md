### Front
* 기초적인 뼈대를 HTML로 작성
* Zeplin을 참조해 CSS적용
* Notion의 기획대로 오브젝트 상호작용을 JS로 구현
* 서버에서 오는 동적데이터 처리를 위한 Thymeleaf이용
